about=GeoEngine provides Docker-based isolated runtimes for geospatial workloads
    with GPU support. This plugin connects QGIS to the GeoEngine proxy service,
    allowing you to run containerized Python and R scripts with GDAL, PyTorch,
    and other geospatial libraries directly from the QGIS Processing toolbox.

    Features:
    - Run GPU-accelerated geoprocessing tools
    - Execute long-running jobs without blocking QGIS
    - Support for custom Python and R scripts
    - Automatic tool discovery from registered projects
